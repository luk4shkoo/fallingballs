# Falling Balls

## Spustenie hry

1. Dvojklikom na súbor `run_game.bat`

2. V príkazovom riadku:
   ```
   javac -d bin src/game/*.java
   java -cp bin game.Main
   ```

## Ovládanie hry
- Pohyb plošiny: šípky **vľavo** a **vpravo**
- Streľba: **medzerník**
- Reštart hry po prehre: **Enter**
